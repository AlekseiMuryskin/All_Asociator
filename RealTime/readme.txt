RecStream.py - выполняет роль ассоциатора в связке c SeisComp4 (на третьем не проверял).
Для того чтобы запустить скрипт:
1) Выполняем скрипты для создания новых таблиц sql в БД SeisComp (MyOrigin и MyPick, порядок не важен)
2) Заполняем конфиги:
    2.1 Конфиг файл config.ini содержит одну секцию Main. В нем необходимо заполнить 4 блока:
      - информацию для допуска к БД: host, user, password и dbase (имя БД)
      - параметра для ассоциатора: название объекта, код сети, окно поиска пиков для объявления события (dt, c),
	минимальное количество станций для объявления события (NSta), ListMode - работать только с конкретными станциями,
	которые перечислены в ListSta через запятую, LogMode - логирование в LogFile
      - параметры STA/LTA: размер буффера в секундах, файл с настройками детектора (STALTAfile)
      - параметры полосового фильтра: нижняя/верхняя частоты, порядок фильтра
    2.2 Файл с параметрами для детектора (stalta.config), поле STALTAfile основного конфига.
	В нем построчно записаны параметры STA/LTA детектора для станций через знак табуляции. STA LTA TrigOn TrigOff Station
3) Можно запускать скрипт: ./seiscomp exec RecStream.py config.ini
